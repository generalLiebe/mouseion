# Mouseion 収益化戦略

## エグゼクティブサマリー

Mouseionは、データ貢献者に公正な報酬を提供しながら、持続可能なエコシステムを構築する収益モデルを目指しています。本ドキュメントでは、段階的な収益化アプローチを説明します。

## 問題の本質

**当初の構想**:
```
推論時ユーザー → 料金支払い → Mouseion手数料 → データ提供者に分配
```

**現実の問題（鶏卵問題）**:
```
データなし → モデルなし → ユーザーなし → 収入なし → 開発継続不可
```

この鶏卵問題を解決するには、マーケットプレイスが完全稼働する前に収益を確立する段階的アプローチが必要です。

### AIデータライセンスで実際にお金が動いている事例

| 事例 | 金額 | モデル |
|:--|:--|:--|
| Shutterstock AIライセンス | $104M/年 | AIデベロッパーへのデータライセンス販売 |
| OpenAI → News Corp | $250M | 大手出版社との直接契約 |
| OpenAI → Axel Springer | $25M | 同上 |
| Google → Reddit | $60M/年 | データ利用料 |

**共通点**: 最初の収入は**B2B契約**から（エンドユーザー課金ではない）

### Story Protocolからの重要な示唆

- オンチェーン取引収入は**ほぼゼロ**（日次約$0）
- 実際の収益は**オフチェーンのライセンス契約**から
- マーケットプレイス手数料は初期収益源として機能しない

   - ブロックチェーンによる出所証明
   - 可逆送金による公正な分配
   - 専門家評価システム

## Phase 2: マーケットプレイス化（12ヶ月〜）

### Phase 1で達成すべき前提条件

- [ ] 1,000人以上のデータ提供者
- [ ] 10社以上のB2B顧客
- [ ] 月間取引額 ¥1,000,000以上
- [ ] 少なくとも1つの学習済みモデルが登録

### マーケットプレイスの収益モデル

```
収入 = データライセンス手数料 + モデルアクセス手数料 + 推論課金

データライセンス手数料: 取引の10-20%
モデルアクセス手数料: 取引の5-15%
推論課金: APIコールの1-5%
```

### 推論時課金（Pay-per-inference）の実装

**Perplexity AI Publisher Programを参考**:
- AIの応答でコンテンツが引用されるたびに支払い
- 使用量ベースで透明性がある

**Mouseion版**:
```
1. ユーザーがモデルにAPIリクエスト
2. Mouseionゲートウェイが利用をログ
3. 日次/週次で集計
4. データ提供者に印税分配（可逆送金）
```

## 重要な意思決定ポイント

### Q1: 最初のデータはどこから調達するか？

**選択肢**:

A. **パブリックドメイン/CC0データ**
   - メリット: 無料、すぐ使える
   - デメリット: 差別化にならない

B. **クリエイターのベータ参加**
   - メリット: オリジナルデータ、コミュニティ形成
   - デメリット: 時間がかかる

C. **提携先のデータ活用**
   - メリット: 大量データを一度に確保
   - デメリット: 提携交渉が必要

**推奨**: B + C の組み合わせ
- 少数のクリエイターで開始（5-20人）
- 同時に提携先を探す（pixiv, Clip Studio等）

### Q2: 最初のB2B顧客は誰か？

**ターゲット優先順位**:
1. 日本のAIスタートアップ（著作権問題で困っている）
2. 大学のAI研究室（論文でデータ出所を示したい）
3. 海外AI企業の日本法人（GDPR/著作権対応）

### Q3: トークンはいつ発行するか？

**推奨: Phase 2以降**

理由:
- 実用性のないトークンは価値が続かない
- 規制リスク（日本では暗号資産規制が厳しい）
- まず実績を作ってからトークン化

## 結論

**「推論時課金で手数料を取る」という最終目標は正しい。**

しかし、そこに至るまでの「つなぎ」が必要。

```
今 → B2Bデータライセンス → マーケットプレイス → 推論時課金
     ↑                      ↑                    ↑
     ここで最初の収入       ここで手数料収入      ここで当初構想実現
```

Shutterstockが最初に「ストックフォト販売」で収益を上げ、後に「AI学習ライセンス」を追加したように、Mouseionも段階的に進化させる。
