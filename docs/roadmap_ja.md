# Mouseion 開発ロードマップ

## 概要

このロードマップは、Mouseionプロトコルの段階的な開発を、初期MVPから完全なエコシステム展開まで概説します。技術開発とビジネス持続性の両方のマイルストーンを統合しています。

詳細な収益化戦略については[収益化戦略](./monetization_ja.md)を参照してください。

---

## Phase 0: 生存期間の確保（今〜3ヶ月）

**フォーカス**: 資金確保と初期実績の構築

### ビジネスマイルストーン

- [ ] **B0.1** グラント申請
  - Web3エコシステムグラント（Cosmos, Polkadot, Web3 Foundation）
  - AI倫理/安全性グラント
  - 国内グラント（NEDO/JST）

- [ ] **B0.2** 初期B2B契約
  - AIスタートアップまたは大学との1-2件のパイロット契約
  - 小規模データセットライセンス（1,000-10,000件）
  - データ出所証明サービス

- [ ] **B0.3** コンサルティング収入
  - データプロバナンス監査
  - 著作権コンプライアンス相談

### 技術マイルストーン

- [ ] **T0.1** 公開リポジトリの整備
  - ドキュメント完成
  - ホワイトペーパー公開
  - 技術仕様公開

- [ ] **T0.2** 基本プロトタイプ
  - コアブロックチェーンのPoC
  - 基本的なトランザクションフローのデモ

### 成功基準
- 月間収入目標: ¥750,000-¥1,500,000
- 少なくとも1件のグラント申請提出
- 少なくとも1件のパイロット契約締結

## Phase 1: 基盤構築（MVP）

**フォーカス**: コアブロックチェーンと基本機能

### マイルストーン

- [ ] **L1.1** シングルノード可逆送金ブロックチェーン
  - 保留/確定状態を持つ基本台帳
  - 取消機能
  - 時間ベースの自動確定

- [ ] **L1.2** 基盤チェーン上でのトークン発行

- [ ] **L1.4** シンプルな報酬分配

- [ ] **L1.5** 基本的なWebウォレット
  - トークン送受信
  - 保留型送金の確認
