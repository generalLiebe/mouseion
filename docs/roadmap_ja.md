# Mouseion 開発ロードマップ

## 概要

このロードマップは、Mouseionプロトコルの段階的な開発を、初期MVPから完全なエコシステム展開まで概説します。

## Phase 1: 基盤構築（MVP）

**フォーカス**: コアブロックチェーンと基本機能

### マイルストーン

- [ ] **L1.1** シングルノード可逆送金ブロックチェーン
  - 保留/確定状態を持つ基本台帳
  - 取消機能
  - 時間ベースの自動確定

- [ ] **L1.2** 基盤チェーン上でのトークン発行
  - ネイティブトークン実装
  - 基本的な送金機能

- [ ] **L1.3** データ登録・メタデータ管理
  - オンチェーンメタデータ保存
  - ハッシュベース検証
  - 権限フラグ（オプトイン/オプトアウト）

- [ ] **L1.4** シンプルな報酬分配
  - 固定比率での分配
  - 手動トリガーメカニズム

- [ ] **L1.5** 基本的なWebウォレット
  - トークン送受信
  - 保留型送金の確認
  - トランザクション履歴

### 成果物
- テストネット展開
- 基本ドキュメント
- 開発者セットアップガイド
