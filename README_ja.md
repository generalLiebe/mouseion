# Mouseion

> AIの価値を、全ての貢献者に公正に還元する

[English README](./README.md)

## 概要

Mouseionは、AIシステムへの貢献を追跡し、データ提供者・クリエイター・レビュワー・AI開発者に対して公正に報酬を分配する分散型プロトコルです。可逆送金型ブロックチェーン上に構築され、透明で監査可能かつ回復可能な支払いフローを実現します。

## 解決する課題

現行のAIシステムには根本的な問題があります：

- **報酬の不在**: AIモデルの学習に貢献したデータ提供者に報酬が還元されていない
- **透明性の欠如**: どのデータがAIの学習・生成に使用されたか追跡できない
- **権利の侵害**: 著作権者の意思に関係なくデータが利用される
- **中央集権的支配**: 少数の企業がAIシステムを独占的に管理している
- **データの喪失**: サービス終了やアカウント削除によりデータが失われる

## 提供する価値

Mouseionが実現すること：

- **データ主権**: データ提供者が自分のデータに対する主権を持てる
- **公正な報酬**: 貢献度に応じた透明な報酬分配
- **民主的ガバナンス**: 貢献者全員がシステム運営に参加できる
- **永続的なデータ保存**: ブロックチェーン上でデータが永続する
- **トレーサビリティ**: 生成物の出典が追跡可能

## 主要機能

### 可逆送金型ブロックチェーン
送金が確定するまで保留状態を維持する新しいLayer 1ブロックチェーン：
- 誤送金からの回復が可能
- ガーディアン審査による詐欺防止
- 安全なマルチパーティ取引

### 専門家アノテーションシステム
- 審査区分表に基づく分野別専門家選定
- 相互レビューによる品質保証
- 評価履歴に基づく専門家マッチング

### 人間データ/AI生成データの分離
- AI生成データによる学習でのモデル崩壊を防止
- 人間が作成したコンテンツの価値を保護
- 明確なタグ付け：「人間作成」「AI生成」「AI支援」

### AIモデルのバージョン管理
- オンチェーンでのバージョン追跡
- サイレントなモデル変更を防止
- ユーザーが特定バージョンを選択可能
