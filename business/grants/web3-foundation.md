## 2. Project Description

### What is Mouseion?

Mouseion is a decentralized protocol that tracks contributions to AI systems and distributes royalties fairly to data providers. It addresses the fundamental problem that creators who provide training data for AI receive no compensation while their work generates enormous value.

### Key Technical Components

1. **Reversible Transaction Blockchain (Layer 1)**
   - Novel consensus mechanism supporting transaction reversibility
   - Pending state management with time-based finalization
   - Guardian system for fraud prevention

2. **AI Contribution Tracking**
   - Semantic similarity analysis (Sentence-BERT)
   - Keyword matching (TF-IDF)
   - RAG-based tracking for generative models

3. **Fair Reward Distribution**
   - Automated royalty calculation
   - Direct distribution to contributor wallets
   - Transparent on-chain accounting

4. **Quality Assurance System**
   - Expert annotation with peer review
   - Human/AI data separation
   - Classification tables for expert matching

### Why Blockchain?

Blockchain provides essential properties for this use case:
- **Immutability**: Provenance records cannot be altered
- **Transparency**: All distributions are publicly verifiable
- **Decentralization**: No single entity controls the system
- **Automation**: Smart contracts enable automatic royalty distribution
